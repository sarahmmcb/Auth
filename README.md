# JWT Auth Server

A .NET Web API for token-based Authentication

## Overview

Provides Token-based authentication, with 2FA and Refresh capabiities

This app is currently in development.

## Features

### User Management
- Contains table for core User info, including First Name, Last Name, User Name, Email

### Role Management
- Currently only supports `Admin` and `User`

### Account Management
- Manages account states and activity record

## Technology Stack

### Backend
- .NET Web API
- SQL Server

## Installation

### Clone the Repository

```bash
git clone https://github.com/example/ce-tracker.git
cd Auth/AuthApi
```

### Install Dependencies

```bash
docker compose up
```
